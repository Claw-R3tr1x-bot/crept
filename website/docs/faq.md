# FAQ

## FAQ
1. **How do I reset my HWID?** Use Dashboard → HWID Reset Request.
2. **Can I change plans?** Yes, from Purchase page or support.
3. **Do you support yearly billing?** Yes with discounts.
4. **How do I contact support?** Use Forms → Help tab.
5. **Where are announcements?** Forms → Announcements tab.
6. **How do I check server health?** Forms → Servers tab.
7. **Can admins ban users?** Yes from Admin → User Management.
8. **Can I export audit logs?** Via admin API endpoint.
9. **Does dark mode persist?** Yes using localStorage.
10. **How to report bugs?** Open a ticket in Forms Help.
11. **How are tokens stored?** HTTP-only cookies.
